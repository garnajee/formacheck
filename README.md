# formacheck

![License](https://img.shields.io/github/license/garnajee/formacheck)

Beautify, format, check syntax code as well as diff mode between 2 files.

Json enhancement thanks to the [Vanilla JSON Editor lib](https://github.com/josdejong/svelte-jsoneditor).

Fully open source, no server, nothing saved, everything is done client-side.
